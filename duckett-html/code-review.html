Code Review

Superscript helps you keep track of dates, such as July 4th.

Subscript is useful for chemical compounds and other items, such as CO2.

The "hr" tag creates a line between separated items (as opposed to a blank space like with "br"). See directly below for example.

Here's a blockquote.

You can also have quotes within text content, for example Blah blah blah, said the cat.

There's even a tag for abbreviations, for example, Prof Stephen Hawking.

There's also an acronym tag, for example, NASA

Cite tag italicizes title names, for example, A Brief History of Time.

There's also a definition tag.

The address tag has quite a specific use: to contain contact details for the author of a page. See example below:

email@email.com

7850 Address Land, Marzipan, MI

You can insert and delete items and communicate that to your readers through tags also. Example: I hate love coding.

The s element indicates something that is no longer accurate or relevant but that should not be deleted. Example: Price, $750. Sale price was $650

The definition list is created with the dl element and usually consists of a series of terms and their definitions. Inside the dl element you will usually see pairs of dt and dd elements. Example is below.
